## 🔍 보험사 타입별 특징

  - 유사암진단비: 600만원 명시
  - 고액암진단비: 롯데/흥국은 1,000만원 명시, 삼성은 미제공

### Type B — 혼합형
- **보험사**: 메리츠, DB손보
- **특징**:
  - 암진단비(유사암제외): 3천만원 명시

  - 고액암진단비: 1천만원 명시

### Type C — Product-level 구조형

- **특징**:

  - **NOTE**: Type C는 "보험가입금액" 방식으로 운영되므로, 담보별 금액이 가입설계서에 명시되지 않는 것이 정상

## 📌 유의사항

1. **금액 명시 없음 (UNCONFIRMED)**
   - 담보는 존재하나 가입설계서에 금액이 명시되지 않은 경우
   - Type C 보험사(한화, 현대, KB)에서 주로 발생
   - 상품 구조상 정상 (보험가입금액 방식)

2. **해당 담보 없음 (NOT_AVAILABLE)**
   - 해당 보험사/상품에 해당 담보가 존재하지 않음
   - 각 보험사의 상품 구성 차이

3. **금액 표기 차이**
   - "3,000만원" vs "3천만원": 표기 방식 차이일 뿐, 동일 금액
   - "600만원" vs "6백만원": 동일
