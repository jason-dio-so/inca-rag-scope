
Design Philosophy
	•	NO LLM (결정 로직)
	•	NO OCR
	•	NO Embedding
	•	Deterministic, Reproducible Pipeline

Core Principles
	1.	Scope-First: 가입설계서 담보만 처리
	2.	Canonical-First: 엑셀 단일 기준
	3.	Evidence-Driven: 모든 비교는 근거 문서 필수
	4.	Linear Pipeline: Step1 → StepN 단방향

Locked Decisions
	•	Step1: Scope + Amount 추출 (고정)
	•	Step2: Canonical Mapping (구조 고정)
	•	Step3~7: Evidence & Compare Pipeline
