## 4. Type 판정 기준

### Type A/B (담보별 금액 명시형)
- ✅ 담보별 가입금액 테이블 존재 (`보장명 | 가입금액`)
- ✅ 각 담보마다 고유 금액 명시 (3천만원, 1천만원, 6백만원 등)
- ✅ 암진단비(유사암제외)에 대해 **3천만원** 명시

### Type C (보험가입금액 참조형)
- ❌ "보험가입금액: X" 단일 값만 존재
- ❌ 모든 담보가 "보험가입금액 지급" 형태로 참조
- ⚠️  일부 담보(상해후유장해 등)는 "보험가입금액" 참조하지만,
  주요 진단비 담보들은 독립적인 금액 보유
