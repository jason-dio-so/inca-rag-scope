## 1. 개요

### 1.1 이 문서의 목적

본 문서는 **디자인 작업이 아닙니다**.
- ✅ Response View Model → UI 렌더링 결과 검증
- ✅ 고객 예제 1~4의 화면 표현 합의
- ✅ 프론트엔드 구현 직전 최종 확인
- ❌ 색상, 폰트, 아이콘 결정 (다루지 않음)

### 1.2 검증 목표

- 회사명 + 상품명 표시 위치 고정
- Evidence 기반 값 표시 일관성
- 보험료 경고 UI 강제 표시
- 추천/추론 표현 0건

### 3.2 회사/상품 선택 UI (미포함)

본 시스템은 **자연어 질문에서 자동 인식**하므로,
별도의 드롭다운/선택 UI는 제공하지 않습니다.

**이유**:
- 질문 예: "삼성생명과 한화생명 건강보험 비교"
- `query_summary.target_products[]` 에서 자동 파싱

## 4. 응답 화면 공통 골격

```
┌─────────────────────────────────────────────────┐
│ [User Message Bubble]                            │

└─────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────┐
│ [System Response Card]                           │

│ └─────────────────────────────────────────────┘ │
│                                                  │
│ ┌─────────────────────────────────────────────┐ │
│ │ [Section 2] query_summary → 비교 대상 상품  │ │
│ └─────────────────────────────────────────────┘ │
│                                                  │
│ ┌─────────────────────────────────────────────┐ │
│ │ [Section 3] comparison → 보장 비교표 (핵심) │ │
│ └─────────────────────────────────────────────┘ │
│                                                  │
│ ┌─────────────────────────────────────────────┐ │
│ │ [Section 4] notes → 요약 및 상세            │ │
│ └─────────────────────────────────────────────┘ │
│                                                  │
│ ┌─────────────────────────────────────────────┐ │
│ │ [Section 5] limitations → 주의사항          │ │
│ └─────────────────────────────────────────────┘ │
│                                                  │
│ [Footer] generated_at: 2025-12-28 10:00         │
└─────────────────────────────────────────────────┘
```

### 4.2 접기/펼치기 정책

| 섹션 | 기본 상태 | 사용자 제어 |
|---|---|---|
| Premium Notice | 항상 펼침 | 불가 (강제 표시) |
| Query Summary | 항상 펼침 | 불가 |
| Comparison | 항상 펼침 | 불가 |
| Notes - summary | 항상 펼침 | 불가 |
| Notes - details | 접힘 | 가능 (펼치기 버튼) |
| Limitations | 접힘 | 가능 (펼치기 버튼) |
