# 고객 피드백 체크리스트

**목적**: UI 프로토타입 검증 및 다음 단계 범위 확정
**작성 방법**: 각 항목에 체크 (☑) 또는 선택 표시
**소요 시간**: 10분 이내

---

## 1. 첫 화면 (Empty State)

### 1.1 시스템 안내 문구

**질문**: "저는 보험 약관과 가입설계서를 기반으로 객관적인 정보를 제공합니다."

- [ ] 적절함 (그대로 사용)
- [ ] 너무 형식적 (더 친근하게)
- [ ] 너무 짧음 (더 상세하게)
- [ ] 기타: _______________

### 1.2 "할 수 있는 것 / 없는 것" 접기/펼치기

**질문**: 기본 상태를 어떻게 할까요?

- [ ] 접힌 상태 (현재 방식)
- [ ] 펼쳐진 상태 (항상 보이기)
- [ ] 아예 제거 (불필요)

### 1.3 예제 버튼 (4개)

**질문**: 예제 버튼이 유용한가요?

- [ ] 유용함 (실제 구현 시에도 유지)
- [ ] 프로토타입에서만 필요 (실제 구현 시 제거)
- [ ] 개수 조정 필요: _____개

## 2. 예제 3 - 상품 종합 비교 (9개 담보)

### 2.1 회사명 + 상품명 표시

**질문**: "📋 비교 대상 상품" 섹션 위치가 적절한가요?

- [ ] 적절함 (현재 위치 유지)
- [ ] 너무 위 (테이블 바로 위로 이동)
- [ ] 너무 아래 (화면 최상단으로 이동)

**질문**: 폰트 크기 / 강조 정도가 적절한가요?

- [ ] 적절함
- [ ] 더 크게
- [ ] 더 작게

### 2.2 9개 담보 비교표

**질문**: 9개 담보가 모두 필요한가요?

- [ ] 필요함 (9개 모두 표시)
- [ ] 너무 많음 (핵심 3~5개만)
- [ ] 접기 기능 추가 (기본 3개 표시 + "더 보기" 버튼)

**질문**: 테이블 가로 스크롤이 불편한가요? (모바일 대응)

- [ ] 불편하지 않음
- [ ] 불편함 (다른 레이아웃 필요)

### 2.3 Evidence (📄 아이콘)

**질문**: 📄 아이콘 위치가 적절한가요?

- [ ] 적절함 (값 아래)
- [ ] 값 옆으로 이동
- [ ] 아이콘 제거하고 텍스트로 표시

**질문**: Evidence 모달에 필요한 정보가 충분한가요?

- [ ] 충분함
- [ ] 부족함 (추가 필요: _______________)
- [ ] 너무 많음 (불필요: _______________)

### 2.4 Evidence 모달 - Snippet

**질문**: Snippet (원문 발췌) 영역이 필요한가요?

- [ ] 필요함
- [ ] 불필요 (문서 출처만 표시)
- [ ] "원문 보기" 버튼만으로 충분

### 2.5 Notes 상세 (7개 항목)

**질문**: Notes 상세 7개 항목이 적절한가요?

- [ ] 적절함
- [ ] 너무 많음 (3~5개로 축소)
- [ ] 더 많아도 됨

**질문**: Notes 기본 상태를 어떻게 할까요?

- [ ] 접힌 상태 (현재 방식)
- [ ] 펼쳐진 상태 (항상 보이기)
- [ ] 요약만 표시하고 상세 제거

### 2.6 주의사항 (Limitations)

**질문**: 주의사항 문구가 적절한가요?

- [ ] 적절함
- [ ] 너무 보수적 (완화 필요)
- [ ] 더 강하게 표시 필요

**질문**: 주의사항 기본 상태를 어떻게 할까요?

- [ ] 접힌 상태 (현재 방식)
- [ ] 펼쳐진 상태 (항상 보이기)

## 3. 예제 1 - 보험료 비교

### 3.1 보험료 경고 배너

**질문**: 노란색 경고 배너가 충분히 눈에 띄나요?

- [ ] 충분함
- [ ] 더 강하게 (빨간색 또는 상단 고정)
- [ ] 너무 강함 (톤 다운 필요)

### 3.2 보험료 표기 방식

**질문**: "약 50,000원 (예시)" 표기가 적절한가요?

- [ ] 적절함
- [ ] "(예시)" 대신 다른 표현: _______________
- [ ] 금액 표시 불필요 (문구만 표시)

### 3.3 보험료 기능 범위

**질문**: 보험료 비교 기능을 제공할까요?

- [ ] 제공함 (현재 방식 - 참고용만)
- [ ] 제공하되 더 강한 경고 필요
- [ ] 제공하지 않음 (기능 제거)
